## 반복문

- 특정 코드 블록을 반복적으로 실행
- 반복 조건 또는 반복 가능한 자료형으로 반복 횟수 제어

## `for` 반복문

- 반복 횟수가 명확할 때 사용
- 초기화, 조건식, 증감식으로 3개의 영역으로 구성
- 실행 코드 블록은 중괄호 `{}`로 구분

```jsx
for (초기화; 조건식; 증감식) {
  // 실행할 코드
}

```

- 초기화: 반복문 시작 전 한 번 실행, 반복문에서 사용할 변수 초기화
- 조건식: 반복 여부를 결정. 평가 결과가 참(true)이면 코드 블록 실행
- 증감식: 코드 블록 실행 후 실행, 조건 변수의 값을 변경하여 반복 제어

### 예시

**5번 반복**

```jsx
for (let i = 0; i < 5; i++) {
  console.log(`현재 숫자: ${i}`);
}

```

```
현재 숫자: 0
현재 숫자: 1
현재 숫자: 2
현재 숫자: 3
현재 숫자: 4

```

**1부터 N까지 반복**

```jsx
let N = 5;

for (let i = 1; i <= N; i++) {
  console.log(`현재 숫자: ${i}`);
}

```

```
현재 숫자: 1
현재 숫자: 2
현재 숫자: 3
현재 숫자: 4
현재 숫자: 5

```

### 중첩 반복문

- 반복문 내부에 다른 반복문을 포함하는 구조

**구구단 예시**

```jsx
for (let i = 2; i <= 9; i++) {
  console.log(`${i}단`);
  for (let j = 1; j <= 9; j++) {
    console.log(`${i} * ${j} = ${i * j}`);
  }
}

```

### block scope

- for문 초기화 식에서 let으로 선언한 변수는 해당 반복문에서만 사용 가능하다. 즉, block scope를 가진다.
- if문과 마찬가지로 for문은 `{} - block` 으로 이루어져있기 때문에 for문 안에서의 변수는 밖에서 사용하지 못한다.
    
    ```jsx
    for (let i = 1; i<5; i++){
      const num = 10;
    }
    
    // console.log(num) // 사용 불가
    // console.log(i)  // 사용 불가
    ```
    
- for문 안에서 변수의 값을 변경해야 하면 미리 let으로 선언해줘야 한다.
    
    ```jsx
    let num = 10;
    
    for (let i = 1; i<5; i++){
      num += i;
    }
    
    console.log(num)
    ```

## `while` 반복문

- 특정 조건이 만족될 때까지 반복해야할 때 사용
- 조건식이 참(true)인 동안 코드 블록 반복 실행
- 잘못된 조건식 작성 시 무한 반복의 위험

**기본 구조**

```jsx
while (조건식) {
  // 실행할 코드
}

```

### 예시

```jsx
let count = 0;

while (count < 5) {
  console.log(`카운트: ${count}`);
  count++;
}

```

```
카운트: 0
카운트: 1
카운트: 2
카운트: 3
카운트: 4

```

### `do...while` 반복문

- `while` 반복문과 동일하지만, 조건문을 실행하기 전 코드블럭을 한번 실행한다.

```jsx
do {
  // 실행할 코드
} while (조건식);

```

## 반복문 제어

### `break`

- 현재 실행 중인 반복문을 즉시 종료

```jsx
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break;
  }
  console.log(`i의 값: ${i}`);
}

```

```
i의 값: 0
i의 값: 1
i의 값: 2
i의 값: 3
i의 값: 4

```

### `continue`

- 현재 반복을 중단하고, 다음 반복으로 진행

```jsx
for (let i = 0; i < 5; i++) {
  if (i === 2) {
    continue;
  }
  console.log(`i의 값: ${i}`);
}

```

```
i의 값: 0
i의 값: 1
i의 값: 3
i의 값: 4

```
