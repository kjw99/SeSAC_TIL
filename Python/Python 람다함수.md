## 람다 함수

- 출처 : https://y00n-lee.tistory.com/25
- python에서 딕셔너리 형태의 데이터를 갖는 리스트가 있을 때, 딕셔너리의 특정 value를 기준으로 정렬하고자 할 때 사용함.
- 람다 함수 자체는 익명 함수라고도 불리고, 이렇게 불리는 이유는 함수이지만, 사용할 때 이름을 부여하지 않기 때문이다.
- 파이썬은 모든 것이 객체이고, 따라서 함수도 객체다. 보통 함수를 만들 때 사용하는 def 키워드는 함수 클래스에 코드와 이름을 담아서 객체를 생성한 뒤, 그 객체를 함수 이름과 동일한 변수에 담아준다. ( 파이썬에서 객체의 타입은 객체를 만든 클래스를 의미하므로 함수 클래스로 만든 객체 타입은 함수이다. )
- 이렇게 함수 객체를 변수에 담으면, 함수 객체가 메모리에 올라가서 변수를 통해 자신이 호출되기를 기다린다. 만약, 만든 함수를 단 한 번만 사용할 것이라면 이는 불필요한 메모리 낭비다.
- 이런 이유로 람다 함수를 사용한다.

```python
c = [[1,2],[3,9],[2,7],[2,1],[2,4],[3,1]]
c.sort(key=lambda x:(x[0],-x[1]))
print(c)
# [[1, 2], [2, 7], [2, 4], [2, 1], [3, 9], [3, 1]]
```

- 위에서 2번째 줄이 람다 함수를 사용한 것!
- 기존의 함수
    - def 함수이름 (매개변수) :
        - return 결과
- 람다 함수
    - lambda 매개변수 : 결과
- 이런식으로 해석하면 된다. 람다 키워드와 콜론 사이의 값은 함수의 매개변수, 콜론 뒤의 값은 결과 값!
- 콜론 뒤에는 하나의 표현식만 들어간다. 여러 요소를 기준으로 정렬하려면 튜플을 사용해서 정렬하고 싶은 순서대로 묶어준다.
    - 내림차순을 원하면 요소 앞에 - 를 붙이면 된다. 그래서 위의 코드에 (x[0], -x[1]) 이 들어간 것.
- 따라서 위 코드의 경우 변수 c의 첫 번째 열 값으로 오름차순, 두번째 열 값으로 내림차순하는 코드가 된다.
- sorted() 함수의 경우 sorted(리스트, key=??) 이런 형식인데, 어떤 값으로 정렬하겠다! 라는 기준값을 만들고 그 기준값을 비교해서 순서를 정하는 형태.
- 람다 함수에서 x의 의미는 리스트 안의 각 요소를 뜻함.
    - for data in datas 에서 data 느낌.
- 아래의 코드 같은 경우 딕셔너리를 값으로 가지고 있는 리스트를 대상으로 정렬하려는 건데, 리스트에 있는 딕셔너리 하나 하나가 x를 뜻함. 그래서 x[”vote”] 라는건 x의 안에 있는 “vote”라는 키의 value 값을 의미하게 된다.

```python
sorted_movies = sorted(
    movies,
    key=lambda x: x["vote"],
    reverse=True
)
```

- 추가로, str.sort() < 이 경우는 본인의 메서드를 사용한 것. sorted(str) < 함수를 사용한 것.
- 함수는 매개변수 데이터를 변경하지 않는 것이 좋음.

```python
# 람다 쓴 것과 안 쓴것 차이
def func(el):
	return el[1]

sorted(data, key=func)
sorted(data, key=lambda el : el[1])
```
