## 모듈

- 모듈(Module)이란, 특정 기능을 하는 코드를 python 파일(.py) 단위로 작성한 것을 말함.
- 모듈을 이용하면 다른 파이썬 파일에 작성된 기능을 가져다 사용할 수 있음. 자바에서 다른 클래스 가져다 쓰듯이
- 모듈 불러오는 방식
    1. import 모듈명
        1. 이후 사용할 때 모듈명.함수명() 이런식으로 사용
    2. 불러올 때 어떤 것을 사용할지 명시하는 방식
        1. from 모듈명 import 함수명
        2. 해당 모듈에 있는 함수를 사용한다고 명시하는 것. 그래서 이후 사용할 때 바로 함수()로 사용 가능.
        3. 함수명에 *을 붙이면 모든 함수 사용이 가능해짐.
